# 版本發布

## 檢查 zipline-tej 版本
若已安裝好zipline-tej，我們可透過以下指令顯示當前zipline-tej的版本。
```python
!pip show zipline-tej
```
為確保當前版本為最新，可以透過以下指令以更新zipline-tej。
```python
!pip install --upgrade zipline-tej
```
